#### 现货24h成交数据

+ url： `/public` (GET)
+ 入参

| 参数key | 参数value       | 备注            |
| :------ | :-------------- | :-------------- |
| command | return24hVolume | 现货24h成交数据 |

+ 出参

  ```json
  {
      "totalETH":"190068.341", // 币种总交易量
      "totalUSDT":"723352589.300", // 总交易额
      "ETH_USDT":{
          "ETH":"190068.341", // 交易量
          "USDT":"723352589.300" // 交易额
      }
  }
  ```
