#### 币种配置

+ url： `/public` (GET)

+ 入参

| 参数key | 参数value        | 备注     |
| :------ | :--------------- | :------- |
| command | returnCurrencies | 币种配置 |

+ 出参

  ```json
  {
      "BTC":{ // 币代号
          "id":2, // 币序号
          "name":"BTC", // 全称
          "txFee":"0.0005", // 提现手续费
          "minConf":"0.001", // 最小充值数量
          "depositAddress":null, // 充值地址
          "disabled":1, // 1展示 0不展示
          "delisted":1 // 0下架 1上架
      }
  }
  ```
