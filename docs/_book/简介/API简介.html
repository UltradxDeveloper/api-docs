API简介

欢迎使用Hoo API！ 你可以使用此 API 获得市场行情数据，进行交易，并且管理你的账户。

在文档的右侧是代码，目前我们仅提供针对 shell 的代码示例。

交易市场分为币币交易和创新区交易。

可以使用以下域名访问：api.hoolgd.com

欢迎有优秀 maker 策略且交易量大的机构参与长期做市商项目。
