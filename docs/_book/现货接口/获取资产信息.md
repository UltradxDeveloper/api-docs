#### 获取资产信息

+ url： `/open/pub/asset` (GET)

+ 入参 (无)

+ 出参

  ```json
  {
      "BTC":{
          "max_withdraw":"1", // 最大提币数量
          "name":"BTC", // 币名称
          "can_withdraw":1, // 1可提币 0不可提币
          "can_deposit":1, // 1可充币 0不可充币
          "unified_cryptoasset_id":2, // 币序号
          "min_withdraw":"0.002" // 最小提币数量
      }
  }
  ```
