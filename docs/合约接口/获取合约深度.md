#### 获取合约深度

+ url： `/markets/contract/orderbook` (GET)
+ 入参

| 参数key      | 参数value | 备注     |
| :----------- | :-------- | :------- |
| contractName | BTC永续   | 合约名称 |
| depth        | 10        | 返回条数 |

+ 出参

  ```json
  {
      "msg":null,
      "code":0,
      "data":{
          "asks":[ // 卖盘
              [
                  "62230.1", // 价格
                  "1.097" // 数量
              ]
          ],
          "bids":[ // 买盘
              [
                  "62115.4", // 价格
                  "0.875" // 数量
              ]
          ],
          "isFrozen":"0", // 冻结数量
          "postOnly":"0",
          "seq":null
      }
  }
  ```
