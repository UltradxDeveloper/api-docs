#### 合约面值信息

+ url： /returncontract (GET)
+ 入参 

| 参数key      | 参数value | 备注     |
| :----------- | :-------- | :------- |
| contractName | BTC永续   | 合约名称 |

+ 出参

  ```json
  {
      "msg":null,
      "code":0,
      "data":{
          "contract_type":"BTC永续", // 合约名称
          "contract_price":"0.001", // 合约每张价格
          "contract_price_currency":"62.2832" // 对应U的价值
      }
  }
  ```
